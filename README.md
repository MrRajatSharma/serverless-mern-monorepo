## setup

1. `yarn`
2. `lerna bootstrap`
3. `cd ./packages/tweetsService && npm run start`
4. `cd ./packages/apolloFederationGW && npm run start`
5. `cd ./packages/frontend && npm start`
6. Open `http://localhost:4000` and access tweets service throught federation GW
